# 🌾 AgriPredict

AgriPredict is a Machine Learning & Deep Learning based web application that assists in smart farming decisions.

It provides:

- 🌱 Crop Recommendation based on soil nutrients and environmental conditions  
- 🌿 Fertilizer Suggestion based on soil composition  
- 🍃 Plant Disease Detection using leaf images  

---

## 🚀 Tech Stack

- Python
- Flask
- Scikit-learn
- PyTorch
- NumPy & Pandas
- HTML, CSS, Bootstrap
- Git

---

## 📊 Features

✔ Predicts the most suitable crop  
✔ Suggests fertilizer improvements  
✔ Detects plant diseases from images
